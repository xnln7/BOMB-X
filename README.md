### $ How to Install

Open the terminal and type following commands.

* `apt update`

* `apt install git -y`

* `apt install python3 -y`

* `pkg install marshal`

* `git clone https://github.com/xnln7/BOMB-X.git`

* `cd BOMB-X`

* `chmod +x *`

  
* `./install`

  
* `python3 bomb.py`
