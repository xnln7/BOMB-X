### $ How to Install

Open the terminal and type following commands.

* `apt update`

* `apt install git -y`

* `apt install python -y`

* `pip install lolcat`

* `git clone https://github.com/EthicalUniverse/Termux-Banner`

* `cd Termux-Banner`

* `chmod +x *`

* `sh install` if not work than use `./install`
